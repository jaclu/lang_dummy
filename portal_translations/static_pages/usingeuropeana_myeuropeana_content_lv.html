Mana Europeana lietošana

Mana Europeana ir tava personīgā vieta Europeana.

Reģistrējies Mana Europeana un iespējams:

  • Saglabāt meklējumu: Tavā rīcībā vienmēr būs aktuālākā informācija par konkrētu tematu Europeana, ja saglabāsi konkrēto vaicājumu, lai vēlāk to varētu atkārtot.
  • Pievienot tagu: Atzīmē savus iemīļotākos Europeana objektus ar sev nozīmīgiem tagiem. Vēlāk apskati visus savus objektus Mana Europeana saskarnē, kārtojot tos pēc sevis izvēlētiem atslēgvārdiem.
  • Saglabāt Mana Europeana: saglabāt jūsu iecienītākās vienības vēlākai aplūkošanai.
  • API atslēgas: Ja esi izstrādātājs, kas reģistrējis vienu vai vairākas API atslēgas, tās būs redzamas API atslēgas cilnī.

Lai saglabātu meklēšanu, pievienotu tagu vai saglabātu vienību, vienkārši pieslēdzies Mana Europeana un klikšķini uz attiecīgu pogu izvēlnē.

Lai apskatītu savas saglabātās vienības, meklējumus un tagus, izmanto Mana Europeana saīsnes galvenē.
