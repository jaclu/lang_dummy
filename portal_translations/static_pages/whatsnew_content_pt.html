O que há de novo

Fizemos algumas alterações: para além de um visual renovado, é agora mais fácil descobrir e explorar os tesouros da Europeana.

Uma das mudanças mais significativas é que o site é agora responsivo. Isto significa que se adapta automaticamente a um conjunto de dispositivos e ecrãs. Quer esteja a utilizar um smartphone de ecrã táctil ou um computador de secretária, o portal ajustará a sua apresentação, de modo a que o layout seja sempre adequado ao ecrã.

Pesquisar é ainda mais fácil com as novas sugestões automáticas que aparecem enquanto digita. Iremos antecipar o que procura e fazer recomendações com base nisso, mesmo antes de terminar de escrever.

A nova página principal realça com mais ênfase os nossos conteúdos, com apresentação de imagens maiores, uma secção de parceiros em destaque, e um carrossel que destaca os nossos pins mais recentes no Pinterest.

Para os programadores, lançámos também a Europeana API (Application Programming Interface) v. 2.0 com consola melhorada e novas funcionalidades. Registe-se para chaves API aqui.

Esperamos que disfrute do novo visual da Europeana.
