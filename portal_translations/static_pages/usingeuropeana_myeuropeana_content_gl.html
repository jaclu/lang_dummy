Uso da Miña Europeana

Miña Europeana é o seu espazo persoal na Europeana.

Rexistre-se Miña Europeana and you can:

  • Gardar esta busca: Manteña-se actualizado sobre un determinado tema na Europeana, salvando unha procura, para que poida facilmente realizar de novo no futuro.
  • Engadir unha etiqueta: Label your favourite items in Europeana with tags that mean something to you. You can then see all objects you have tagged, sorted by the keywords you have chosen, in the My Europeana interface.
  • Gardar na Miña Europeana: Garda os seus elementos favoritos para ver máis tarde.
  • API keys: If you are a developer who has registered for one or more API keys, these will be shown on the API Keys tab.

Para gardar unha procura, engadir unha etiqueta ou gardar un elemento, basta con acceder a Miña Europeana and click on the relevant action from the menu.

To view your saved items, searches and tags, use the Miña Europeana shortcuts in the header.
