'My Europeana'

Моя Europeana - это Ваше личное пространство на Europeana.

Зарегистрируйтесь на Моя Europeana и Вы сможете:

  • Сохранить результаты поиска: Быть в курсе новых объектов, связанных с конкретной темой, поступающих в Europeana; сохраните параметры поиска, чтобы провести такой же поиск еще раз в будущем.
  • Добавить тэг: Пометьте Ваши любимые объекты на Europeana тэгами, которые что-то значат для Вас. Затем Вы сможете посмотреть на My Europeana все объекты с Вашими тэгами, отсортированные по ключевым словам, которые Вы выбрали для тэгов.
  • Сохранить в Моей Europeana: Сохраните свои любимые объекты, чтобы посмотреть их позднее.
  • Коды API: Если Вы разработчик, зарегистрированный для получения одного или более кодов API, это будет видно в таблице кодов API

Чтобы сохранить параметры поиска, добавить тэг или сохранить найденный объект, просто войдите с паролем в Моя Europeana и нажмите на нужную Вам позицию меню.

Для того, чтобы просмотреть сохраненные объекты, параметры поисков и объекты с тэгами, используйте подсказки наверху страницы Моя Europeana
