Was ist neu

Wir haben einige Veränderungen vorgenommen, so dass Europeana nicht nur ein neues Aussehen hat, sondern es für Sie auch leichter ist, Europeanas Schätze zu entdecken.

Eine der größten Veränderungen ist, dass die Seite nun reagiert. Das heißt, dass sie sich einer ganzen Reihe von Endgeräten und Bildschirmgrößen automatisch anpasst. Ob Sie ein Touchscreen, ein Smartphone oder einen PC benutzen, das Portal wird seine Anzeige anpassen, so dass das Layout stets zum Bildschirm passt.

Die Suche ist jetzt dank neuer automatischer Wortvorschläge, die erscheinen, wenn Sie Ihren Suchbegriff eingeben, noch leichter. Wir vermuten, was Sie suchen und machen Begriffsvorschläge, noch bevor Sie Ihre Suchbegriffe fertig eingegeben haben.

Die neue Homepage legt einen größeren Schwerpunkt auf kuratierte Inhalte und bietet eine umfassendere Bilder-Diashow, einen Bereich für Partnereinrichtungen im Fokus und ein Bilder-Karussell, das die neuesten Beiträge auf Pinterest zeigt.

Für die Entwickler unter Ihnen haben wir außerdem das Europeana API v2.0 mit einer verbesserten Konsole und neuen Funktionen freigegeben. Registrieren Sie sich für API-Schlüssel hier.

Wir hoffen, dass Ihnen die neue Europeana gefällt.
