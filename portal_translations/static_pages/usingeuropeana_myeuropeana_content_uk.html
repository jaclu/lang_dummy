Використовуючи Моя Європіана

Моя Європіана це ваш персональний простір на Європіана.

Зареєструватись у Моя Європіана та ви можете:

  • Зберегти цей пошук: Будьте в курсі вибраної теми в Європіана зберігши пошук, так що ви можете легко виконати його знову в майбутньому.
  • Додати таг: Дайте назву об'єктам, що вам до вподоби в Європіана за допомогою тагів, що мають для вас значення. Після цього ви зможете побачити всі об'єкти, що ви назвали, відсортованими за ключевим словом, що ви обрали, в інтерфейсі Моя Європіана.
  • Зберегти у Моя Європіана: Збережіть ваші улюблені об'єкти для подальшого просмотру.
  • API ключі: Якщо ви розробник, що зареєструвався, щоб отримати один чи більше API ключів, це буде відображено на вкладці із API ключами.

Щоб зберегти пошук, додати таг або зберігти об'єкт, лише увійдіть у систему Моя Європіана та оберіть необхідну дію із меню.

Для перегляду об'єктів, що збережено та тагів, використовуйте Моя Європіана короткі посилання в заголовку
