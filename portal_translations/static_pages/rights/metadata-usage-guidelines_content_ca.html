Directrius d'Europeana d'ús de les metadades

Aquestes directrius d'us es basen en la bona voluntat, no són un contracte legal, però Europeana us demana que les seguiu si utilitzeu metadades d'Europeana.

Totes les metadades publicades per Europeana són disponibles sense restriccions sota la Creative Commons CC0 1.0 Universal Public Domain Dedication. No obstant, Europeana us demana que reconegueu l'autoria i feu agraïment actiu a les fonts de les metadades, com són els proveïdors de dades (quan són una institució del patrimoni cultural específica) i els agregadors de dades, inclosa Europeana.

Atorgueu el mèrit a qui correspongui.

  • Feu el reconeixement d'autoria als proveïdors de dades i als agragadors, inclosa Europeana. Els agregadors fan una feina crucial en recollir, emmagatzemar i harmonitzar les dades, de manera que siguin àmpliament accessibles i interoperables.
  • Assegureu-vos que els altres siguin conscients de l'estatus de l'estat dels drets de les metadades d'Europeana i d'aquestes directrius. Una manera de fer això és mantenir els enllaços la CC0 Public Domain Dedication, intactes.
  • Si, per raons tècniques o d'altre tipus, no podeu incloure tots els enllaços a les fonts i a la informació de drets, juntament amb les metadades que heu utilitzat, considereu d'incloure'ls per separat, per exemple, en un document a banda que es distribueixi amb les metadades o el conjunt de dades.
  • Si, per raons tècniques o d'altre tipus, no podeu incloure tots els enllaços a les fonts i a la informació de drets, juntament amb les metadades que heu utilitzat, considereu enllaçar només a la font de metadades específica a Europeana, en la qual es troben disponibles les fonts i informació de drets, inclosa la versió en format llegible per màquina.

Les metadades són dinàmiques; considereu utilitzar-les via API o enllaç a Europeana.

  • Quan treballeu amb metadades obtingudes via Europeana, sigueu conscients que no és estàtica i està subjecte a canvis. Europeana actualitza continuament les seves metadades per corregir errors i incloure informació addicional. Per tant, la millor manera d'utilitzar les metadades publicades per Europeana és a través de l'API d'Europeana o mitjançant l'enllaç a les dades específiques. Si, per raons tècniques o d'altre tipus, no podeu fer això, proporcioneu un enllaç a la font de les dades d'Europeana o implementar un mecanisme d'actualització.

Indiqueu les modificacions que realitzeu de les metadades i feu les vostres metadades accessibles sota les mateixes condicions.

  • Si us plau, no utilitzeu les metadades de manera que suggereix que tenen caràcter oficial o que el proveïdor de dades, agregador o Europeana us dóna suport o dóna suport a l'ús que feu de les metadades, llevat que tingueu una autorització prèvia per fer-ho.
  • Si us plau, no confongueu als altres o tergiverseu les metadades o les seves fonts.
  • Si us plau, feu accessibles les metadades i les seves millores en les mateixes condicions que Europeana, és a dir, sense reclamar cap dret legal, o imposar cap condició legalment vinculant sobre l'accés a les metadades i les seves millores.

Tingueu en compte que feu ús de les metadades sota la vostra responsabilitat.

  • Europeana ofereix les metadades tal qual, no les repesenta ni dona garanties de cap tipus pel que fa a les metadades que es publiquen a Europeana.
  • L'ús de les metadades ha de respectar la legislació i regulacions vigents a la vostra jurisdicció, especialment, encara que no únicament, pel que fa a la protecció de dades, difamació i drets d'autor.
  • Sigueu conscients de les normes addicionals que són d'aplicació a les comunitats dels proveïdors i agregadors de dades.
