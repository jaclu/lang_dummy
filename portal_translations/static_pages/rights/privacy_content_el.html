Προστασία ιδιωτικής ζωής

Τί πληροφορίες συλλέγουμε;

Σύμφωνα με την κοινή πρακτική, όταν επισκέπτεστε το δικτυακό τόπο Europeana και όταν εγγράφεστε σε αυτόν, καταγράφουμε και αποθηκεύουμε ορισμένες πληροφορίες για σας. Σε ορισμένες περιπτώσεις προωθούμε αυτές τις πληροφορίες προκειμένου να κατανοήσουμε καλύτερα τους χρήστες μας και τις ανάγκες τους. Αυτό συμβαίνει για να βελτιώσουμε το περιεχόμενο και τη χρηστικότητα της δικτυακής πύλης μας, αλλά και τις υπηρεσίες που προσφέρουμε. Όμως η Europeana.eu αναγνωρίζει τη σπουδαιότητα προστασίας της ιδιωτικής ζωής για τους χρήστες του δικτυακού τόπου Europeana.eu. Συνεπώς μεταχειριζόμαστε με φροντίδα τέτοιες προσωπικές πληροφορίες.

Παρακαλείσθε να διαβάσετε τη Δήλωσή μας για την Προστασία της Ιδιωτικής Ζωής για να μάθετε τί είδους πληροφορίες καταχωρούμε και πώς τις χρησιμοποιούμε. Συλλέγουμε τα ακόλουθα: αρχεία καταχωρήσεων, «μπισκότα» και πληροφορίες που ζητούμε ρητά να μας παρέχετε για τον εαυτό σας κατά την εγγραφή ή κατά τη χρήση κάθε ηλεκτρονικής μορφής επικοινωνίας με το προσωπικό της Europeana.eu.

Αρχεία Καταχωρήσεων και "Μπισκότα"

Όταν επισκέπτεστε τη Europeana.eu, τα ερωτήματα και οι ενέργειές σας καταγράφονται αυτόματα στο διακομιστή μας. Τα αρχεία καταχωρήσεων περιέχουν πληροφορίες όπως η IP διεύθυνσή σας, η ημερομηνία και ο χρόνος κατά τον οποίο είχατε πρόσβαση στο δικτυακό τόπο, οι σελίδες που ζητήσατε να δείτε κ.λπ. Τα αρχεία καταχωρήσεων δεν περιέχουν προσωπικές πληροφορίες που μπορούν να σας ταυτοποιήσουν.

Τα "μπισκότα" είναι μικρές ποσότητες πληροφοριών που στέλνονται από ένα δικτυακό διακομιστή στο φυλλομετρητή σας και αποθηκεύονται στο σκληρό δίσκο του υπολογιστή σας. Περιέχουν πληροφορίες για τους δικτυακούς τόπους που έχετε επισκεφτεί. Η Europeana.eu χρησιμοποιεί «μπισκότα» για να αναγνωρίσει εσάς και τα προνόμια πρόσβασής σας. Οι περισσότεροι φυλλομετρητές ιστού αποδέχονται αυτόματα τα «μπισκότα», αλλά συνήθως μπορείτε να αλλάξετε τις ρυθμίσεις του φυλλομετρητή σας για να τα εμποδίσετε (αν και ίσως δεν έχετε πρόσβαση στην πλήρη λειτουργία του δικτυακού τόπου, όπως οι ρυθμίσεις γλώσσας που ίσως έχετε επιλέξει).

Η Europeana.eu συλλέγει ή καταχωρεί «μπισκότα» και πληροφορίες για τους χρήστες της και επεξεργάζεται αυτές τις πληροφορίες για να παράγει στατιστικές χρήσης. Με την επεξεργασία αυτών των πληροφοριών, προσπαθούμε να προσδιορίσουμε την εικόνα των χρηστών και τις συνήθειες χρήσης για να βελτιώσουμε το περιεχόμενο και τις υπηρεσίες που προσφέρουμε, σύμφωνα με τις ανάγκες των χρηστών. Η ανάλυση των αρχείων καταχωρήσεων χρησιμοποιείται επίσης για να πληροφορούμε τις βιβλιοθήκες-εταίρους της Europeana.eu για τις προτεραιότητες ψηφιοποίησης με βάση τις προτιμήσεις των χρηστών. Αυτά τα συσσωρευτικά στοιχεία δεν περιέχουν πληροφορίες από τις οποίες ταυτοποιείται ένα άτομο. Χρησιμοποιούνται αυστηρά για τους πιο κάτω σκοπούς:

  1. για τον έλεγχο και την καλύτερη διαχείριση της κυκλοφορίας στο δικτυακό τόπο
  2. για την αποτίμηση και τη βελτίωση της χρηστικότητας, των υπηρεσιών και του περιεχομένου
  3. για τον εντοπισμό τεχνικών ελλείψεων και για την αποκατάσταση βλαβών
  4. για την ανάπτυξη εξατομικευμένων υπηρεσιών
  5. για την ανάπτυξη αποτελεσματικών εργαλείων μάρκετινγκ

Δεδομένα εγγεγραμμένων χρηστών και δεδομένα από άλλα ηλεκτρονικά έντυπα

Όταν σας ζητάμε να εγγραφείτε, σας ζητάμε να μας παρέχετε εθελοντικά ορισμένες προσωπικές πληροφορίες για τον εαυτό σας. Η ταυτοποίηση των χρηστών μάς επιτρέπει να παρέχουμε εξατομικευμένες υπηρεσίες στη δικτυακή πύλη. Προς το παρόν προσφέρονται οι ακόλουθες επιλογές: να αποθηκεύετε τα «αγαπημένα» ή την ιστορία της πορείας έρευνας για να αναφερθείτε σε αυτά μεταγενέστερα. Σκοπεύουμε να επεκτείνουμε αυτές τις υπηρεσίες για να προσφέρουμε πιο εξατομικευμένες δυνατότητες στους χρήστες, όπως: αποθήκευση επιλεγμένων συλλογών για μεταγενέστερη χρήση ή παραλαβή ενημερωτικών μηνυμάτων ηλ-ταχυδρομείου για νέες συλλογές που προστίθενται και που ίσως τους ενδιαφέρουν.

Μέσω της χρήσης κάθε ηλεκτρονικού εντύπου στο δικτυακό τόπο μας (έντυπο ανάδρασης, συνδρομή σε ενημερωτικό δελτίο, αναφορά σύνδεσης) συμφωνείτε να παρέχετε τις ελάχιστες προσωπικές πληροφορίες (δηλαδή όνομα και διεύθυνση ηλ-ταχυδρομείου). Ίσως χρησιμοποιήσουμε αυτές τις πληροφορίες για να σας απαντήσουμε σχετικά με το ζήτημα που σας απασχολεί. Οι προσωπικές σας πληροφορίες δεν θα χρησιμοποιούνται για άμεση αλληλογραφία ή για άλλους λόγους πέραν των απαντήσεων στα ερωτήματά σας.

Προστασία Δεδομένων

Η Europeana.eu σέβεται την ιδιωτική ζωή σας και επιδιώκει να προστατεύει τα δεδομένα προσωπικού χαρακτήρα. Δεν πουλάμε, δεν εμπορευόμαστε ή δεν ενοικιάζουμε σε άλλους τις πληροφορίες που παίρνουμε από εσάς σε αυτόν το δικτυακό τόπο. Ορισμένες φορές συνεργαζόμαστε με δημόσιους οργανισμούς, όπως πανεπιστήμια, που διενεργούν έρευνες για λογαριασμό μας σχετικά με πρότυπα χρήσης και συμπεριφοράς για να μας βοηθήσουν να βελτιώσουμε το περιεχόμενο και τις υπηρεσίες που προσφέρουμε. Στις περιπτώσεις αυτές δεν τους αποκαλύπτουμε προσωπικές πληροφορίες (π.χ. ονόματα ή διευθύνσεις).

Η Europeana.eu λαμβάνει όλα τα απαραίτητα μέτρα για την προστασία των δεδομένων από μη εξουσιοδοτημένη πρόσβαση και χρήση με τρόπο άλλον από αυτόν που δηλώνεται στην Πολιτική Προστασίας της Ιδιωτικής Ζωής. Η Europeana.eu φιλοξενείται από το SARA στην Ολλανδία και επομένως χρησιμοποιεί τα ίδια συστήματα και την ασφάλεια που ισχύουν σε αυτό το δημόσιο ίδρυμα.

Τα δεδομένα αποθηκεύονται για όσο χρόνο είναι χρήσιμα για διοικητικούς λόγους. Μη προσωπικά δεδομένα μπορούν να αποθηκεύονται επ’ αόριστον ως ερευνητικό υλικό.

Παρακαλούμε σημειώστε: σύμφωνα με το νόμο, ίσως απαιτηθεί να αποκαλύψουμε στις αρμόδιες νομικές αρχές τις πληροφορίες που καταχωρούμε.

Σχετικά με τη Δήλωση για την Πολιτική Προστασίας της Ιδιωτικής Ζωής

Αυτός ο δικτυακός τόπος ίσως περιέχει συνδέσεις σε άλλους δικτυακούς τόπους που βρίσκονται πέρα από τον έλεγχό μας. Η πολιτική προστασίας της ιδιωτικής ζωής εφαρμόζεται μόνο για τις πληροφορίες που παρέχετε κατά τη διάρκεια της επίσκεψής σας σε αυτόν το δικτυακό τόπο. Άλλοι δικτυακοί τόποι με τους οποίους συνδέεστε, ίσως περιέχουν πολιτικές προστασίας της ιδιωτικής ζωής, που είναι διαφορετικές από την παρούσα πολιτική προστασίας της ιδιωτικής ζωής.

Η τρέχουσα δήλωση για την πολιτική προστασίας της ιδιωτικής ζωής είναι σύμφωνη με τον Ολλανδικό Νόμο περί Προστασίας Δεδομένων Προσωπικού Χαρακτήρα (Wet Bescherming Persoonsgegevens).

Οι αλλαγές στην πολιτική της Europeana.eu για την προστασία της ιδιωτικής ζωής θα δημοσιεύονται σε αυτήν τη σελίδα.

Ερωτήσεις, Παράπονα και Επίλυση Διαφορών

Με τη χρήση του δικτυακού τόπου μας συναποδέχεστε τους όρους για τη Δήλωση Προστασίας της Ιδιωτικής Ζωής που περιγράφεται εδώ. Όμως διατηρείτε το δικαίωμα να εμποδίσετε τη χρήση των προσωπικών σας δεδομένων για εξατομικευμένες υπηρεσίες και μάρκετινγκ, καθώς και να ζητήσετε να μην επικοινωνούμε πλέον μαζί σας. Αν επιθυμείτε να πληροφορηθείτε για τα προσωπικά δεδομένα που έχουν συλλεχθεί για σάς ή να ζητήσετε τη διόρθωση ή την απομάκρυνση αυτών των δεδομένων, ή αν έχετε άλλες ερωτήσεις σχετικές με αυτή τη δήλωση, παρακαλείσθε να επικοινωνήσετε με τη Europeana.eu.

Τελευταία τροποποίηση: 30 Σεπτέμβριος 2008

Έντυπο Ανάδρασης

Αν αντιμετωπίζετε πρόβλημα από τη χρήση του ιστοτόπου ή αν επιθυμείτε να υποβάλετε γενικά σχόλια/προτάσεις, παρακαλείσθε να χρησιμοποιήσετε αυτό φόρμα επικοινωνίας.

Όλα τα ηλεκτρονικά έντυπά μας που απαιτούν να εισάγετε προσωπικές πληροφορίες, είναι σχεδιασμένα σύμφωνα με τις απαιτήσεις για την προστασία της ιδιωτικής ζωής. Αυτή η πολιτική προστασίας της ιδιωτικής ζωής παρέχει περαιτέρω λεπτομέρειες.

Οι μεταφράσεις έχουν γίνει σε εθελοντική βάση από τους συνεργάτες μας και είναι μόνο για ενημέρωση. Το νομικά έγκυρο κείμενο είναι μόνο στην αγγλική γλώσσα.
