Κατευθυντήριες Οδηγίες Χρήσης για Μεταδεδομένα της Europeana

Αυτές οι κατευθυντήριες οδηγίες χρήσης βασίζονται στην καλή διάθεση. Δεν αποτελούν νομικό συμβόλαιο, αλλά η Europeana σας ζητά να ακολουθείτε τις κατευθυντήριες αυτές οδηγίες, αν χρησιμοποιείτε μεταδεδομένα από τη Europeana.

Όλα τα μεταδεδομένα που δημοσιεύει η Europeana είναι διαθέσιμα ελεύθερα περιορισμών κάτω από Creative Commons CC0 1.0 Universal Public Domain Dedication. Όμως η Europeana ζητά να αναγνωρίσετε ενεργά και να αποδώσετε τα εύσημα σε όλες τις πηγές μεταδεδομένων, όπως τους παρόχους δεδομένων (π.χ. ένα εξειδικευμένο ίδρυμα πολιτιστικής κληρονομιάς) και σε κάθε συναθροιστή δεδομένων όπως η Europeana.

Δώστε τα ονόματα των συντελεστών όπου απαιτείται

  • Αποδώστε τα εύσημα στον πάροχο δεδομένων και σε όλους τους συμβαλλόμενους συναθροιστές δεδομένων, όπως η Europeana. Οι συναθροιστές επιτελούν κρίσιμο έργο στη συλλογή, αποθήκευση και εναρμόνιση δεδομένων, έτσι ώστε να είναι ευρέως προσβάσιμα και διαλειτουργικά.
  • Βεβαιωθείτε ότι οι άλλοι είναι ενήμεροι για την κατάσταση των πνευματικών δικαιωμάτων των μεταδεδομένων της Europeana και ότι είναι ενήμεροι για τις κατευθυντήριες αυτές οδηγίες. Ένας τρόπος για να κάνετε αυτό είναι να διατηρείτε συνδέσμους στο CC0 Ανέπαφη Αφοσίωση στο Δημόσιο Τομέα (Public Domain Dedication Intact).
  • Αν, για τεχνικούς ή άλλους λόγους, δεν μπορείτε να περιλάβετε όλους τους απαραίτητους συνδέσμους πηγών και τις πληροφορίες πνευματικών δικαιωμάτων μαζί με τα μεταδεδομένα που έχετε χρησιμοποιήσει, πρέπει να σκεφθείτε να τα περιλάβετε χωριστά, για παράδειγμα σε χωριστό έγγραφο που διανέμεται με τα μεταδεδομένα ή το σύνολο δεδομένων.
  • Αν, για τεχνικούς ή άλλους λόγους, δεν μπορείτε να περιλάβετε όλους τους απαραίτητους συνδέσμους πηγών και τις πληροφορίες πνευματικών δικαιωμάτων μαζί με τα μεταδεδομένα που έχετε χρησιμοποιήσει, μπορείτε να σκεφθείτε να τα συνδέσετε μόνο με τη συγκεκριμένη πηγή μεταδεδομένων της Europeana, όπου βρίσκονται όλες οι διαθέσιμες πηγές και πληροφορίες πνευματικών δικαιωμάτων, περιλαμβάνοντας μηχαναγνώσιμες μορφές.

Τα μεταδεδομένα είναι δυναμικά. Σκεφθείτε να χρησιμοποιήσετε τα μεταδεδομένα μέσω των API της Europeana ή με συνδέσμους.

  • Όταν εργάζεστε με μεταδεδομένα που αποκτάτε από τη Europeana, παρακαλείσθε να είστε ενήμεροι ότι δεν είναι στατικά, αλλά ενδέχεται να αλλάξουν. Η Europeana ανανεώνει συνεχώς τα μεταδεδομένα της για να διορθώνει λάθη και περιλαμβάνει νέες και πρόσθετες πληροφορίες. Επομένως, ο καλύτερος τρόπος χρήσης των μεταδεδομένων που δημοσιεύει η Europeana είναι μέσω των API της Europeana ή με σύνδεσμο σε εξειδικευμένα δεδομένα. Αν, για τεχνικούς ή άλλους λόγους, δεν μπορείτε να το κάνετε, τότε να δημιουργήσετε σύνδεσμο στην πηγή των δεδομένων στη Europeana ή να εφαρμόσετε ένα μηχανισμό ανανέωσης.

Αναφέρετε τις τροποποιήσεις των μεταδεδομένων και διαθέστε τα τροποποιημένα μεταδεδομένα με τους ίδιους όρους.

  • Παρακαλείσθε να μη χρησιμοποιείτε τα μεταδεδομένα με τρόπο που υπονοεί ότι έχετε επίσημη θέση ή ότι ο πάροχος δεδομένων, ο συναθροιστής ή η Europeana σας εγκρίνει τη χρήση των μεταδεδομένων, εκτός αν έχετε προηγούμενη άδεια να ενεργήσετε με τον τρόπο αυτό.
  • Παρακαλείσθε να μην παραπλανείτε άλλους ή να διαστρεβλώνετε τα μεταδεδομένα ή τις πηγές τους.
  • Παρακαλείσθε να διαθέτετε ελεύθερα τα μεταδεδομένα και όποιες βελτιώσεις με όρους ίδιους με της Europeana, π.χ. χωρίς απαίτηση νομικών δικαιωμάτων ή επιβολή νομικών συνθηκών δεσμεύσεων στην πρόσβαση σε μεταδεδομένα ή στις βελτιώσεις σας.

Παρακαλείσθε να σημειώσετε ότι χρησιμοποιείτε τα μεταδεδομένα με δική σας ευθύνη.

  • Η Europeana προσφέρει τα μεταδεδομένα όπως είναι και δεν προβαίνει σε διαβήματα ή εγγυήσεις κάθε είδους σχετικά με τα μεταδεδομένα που δημοσιεύει η Europeana.
  • Κάθε χρήση μεταδεδομένων πρέπει να συμμορφώνεται με όλη την ισχύουσα νομοθεσία και άλλους κανονισμούς στη δικαιοδοσία σας, ιδιαίτερα σχετικά με (χωρίς να περιορίζεται) προστασία δεδομένων, δυσφήμιση ή πνευματικά δικαιώματα.
  • Προσέξτε και κάθε πρόσθετους ισχύοντες κανόνες της κοινότητας των παρόχων δεδομένων και των συναθροιστών δεδομένων.
