Доступність

Функції та контент Europeana.eu було створено, щоб він був універсально доступним користувачам та іх пристроям. Ми намагаємось слідувати стандартам WC3 Web Content Accessibility Guidelines Version 1.0.

Наша команда Централізованого Дизайну Користувачів регулярно проводить юзабіліті тестування Europeana.eu навігації, потоку, заголовків та логічності. Окрім того, ми створимо версію – розповсюдити цю фазу прототипу – для яскравих людей.

Деякі функції доступності нашого сайту включають:

  • Змістовні заголовки сторінок задля допомоги із орієнтацією
  • Майже всі зображення мають змістовний alt-текст (текстовий єквівалент)
  • Мінімальне використання фреймів
  • Використання шрифтів із змінним розміром
  • Використовуємо Cascading Style Sheets (CSS) для розділення контенту від презентацій
  • Повідомити користувача, коли посилання відчиняється у новому вікні
  • Ядро функціональностей можна використати без JavaScript

Будьте ласкаві надішліть нам будь яку інформацію чи питання про доступність Europeana.eu разом із форма зворотнього зв'язку.
