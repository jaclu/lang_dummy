Europeana Brugervejledning for metadata

Disse brugervejledninger er baseret på goodwill og er ikke en retlig bindende aftale, men Europeana henstiller til at du følger disse vejledninger, hvis du bruger metadata fra Europeana.

Al metadata, som Europeana publicerer, kan uden begrænsning bruges under Creative Commons CC0 1.0 Universal Public Domain Dedication. Europeana henstiller imidlertid til, at du aktivt anerkender og tilskriver alle kilder til metadata deres ophav såsom dataleverandører (f. eks. en bestemt kulturarvsinstitution) og data-aggregatorer, herunder også Europeana

Brug kildeangivelser, hvor de bør fremgå.

  • Brug kildeangivelse (dataleverandører og alle bidragydende aggregatorer, inkl. Europeana). Aggregatorer påtager sig en afgørende rolle ved at samle, gemme og harmonisere data, således at de er tilgængelige og indbyrdes kompatible.
  • Sørg for at andre, som bruger metadata på Europeana, bliver bekendt med rettighedsstatus til denne metadata og kender disse vejledninger. En måde at gøre dette er at holde links til CC0 Public Domain Dedication intakte.
  • Hvis du af tekniske eller andre grunde ikke kan gøre brug af de givne links til kilder og oplysninger om rettigheder samt den søgte metadata, kan du udskille dem separat, f.eks. i et selvstændigt dokument, der følger med de givne data eller metadata.
  • Hvis du af tekniske eller andre grunde ikke man gøre brug af de givne links til kilder og informationer om rettigheder samt den søgte metadata, kan du eventuelt vælge kun at linke til den specifikke kilde til metadata på Europeana, hvor du finder alle tilgængelige kilder og information om rettigheder, herunder maskin-læsbare formater.

Metadata er dynamiske. Brug eventuelt metadata via Europeanas API eller via links.

  • Når du arbejder med metadata via Europeana.eu, skal du være opmærksom på, at dette ikke er statisk materiale, men undergår ændringer. Europeana opdaterer løbende metadata for at rette fejl og tilføje ny og udvidet information. Den bedste måde at bruge metadata fra Europeana er derfor via Europeana API eller ved at linke til de specifikke data. Hvis det af tekniske eller andre grunde ikke er muligt, kan du søge et link til kilden bag de pågældende data via Europeana eller benytte en opdateringsmekanisme.

Anfør dine ændringer i metadata og gør den ændrede metadata tilgægelig under de samme vilkår.

  • Undlad venligst at anvende metadata på en måde, der antyder en eller anden form for officiel status, eller at bidragyder, aggregator eller Europeana godkender dig eller din brug af metadata uden forudgående tilladelse.
  • Undlad at vildlede andre eller forvanske metadata eller datakilder.
  • Gør venligst metadata og enhver forbedring heraf frit tilgængelige under samme vilkår som Europeana, dvs. uden krav om nogen juridisk ret eller pålæg om juridisk bindende betingelser ved adgang til metadata eller til eventuelle forbedringer.

Bemærk, at brug af metadata sker på eget ansvar.

  • Europeana tilbyder metadata som det foreligger og står ikke inde for eller giver nogen garantier med hensyn til metadata fremlagt af Europeana.
  • Enhver brug af metadata skal overholde alle gældende love og andre regulativer inden for din jurisdiktion, herunder navnlig (men ikke begrænset til) databeskyttelse, injurier eller ophavsret.
  • Vær opmærksom på evt. andre anvendelige regler inden for fællesskabet, der måtte være gældende for bidragydere og aggregatorer af data.
