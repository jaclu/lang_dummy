Accesibilidade

As funcionalidades e o contido de Europeana.eu deseñáronse de modo que sexan accesibles para todos os usuarios, e todos os programas/dispositivos que utilicen. Esforzámonos por cumprir as Pautas de Accesibilidade ao Contido na Web do WC3 - versión 1.0.

O noso equipo de Deseño Centrado no Usuario leva a cabo frecuentes probas de usabilidad para avaliar a navegación, os fluxos, as etiquetas, e a coherencia. Ademais, produciremos unha versión - tras a fase de prototipo - para persoas cegas.

Algúns dos recursos de accesibilidade do noso sitio web inclúen:

  • Títulos de páxina significativos, para axudar a orientarse
  • Case todas as imaxes teñen tamén un texto alternativo significativo (texto equivalente)
  • Utilización mínima de marcos
  • Utilización de fontes escalables
  • Utilización de Follas de Estilo en Fervenza (CSS-Cascading Style Sheets ) para separar o contido da presentación
  • Notificación aos usuarios cando se abre un enlace nunha nova fiestra
  • As funcionalidades (básicas) poden utilizarse sen JavaScript
