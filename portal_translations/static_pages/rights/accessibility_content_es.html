Accesibilidad

Las funcionalidades y el contenido de Europeana.eu se han diseñado de modo que sean accesibles para todos los usuarios, y todos los programas/dispositivos que utilicen. Nos esforzamos por cumplir las Pautas de Accesibilidad al Contenido en la Web del WC3 - versión 1.0.

Nuestro equipo de diseño centrado en el usuario realiza con frecuencia tests de usabilidad en Europeana.eu, analizando la navegación, los flujos de información, los etiquetados y la coherencia. Es más, tras esta fase de prototipo, se creará una versión adaptada para invidentes.

Entre las funcionalidades de accesibilidad que ofrece nuestro sitio web se incluyen :

  • Títulos de página significativos, para ayudar a orientarse
  • Casi todas las imágenes tienen también un texto alternativo significativo (texto equivalente)
  • Utilización mínima de marcos
  • Utilización de fuentes escalables
  • Utilización de Hojas de Estilo en Cascada (CSS) para separar el contenido de la presentación
  • Notificación a los usuarios cuando se abre un enlace en una nueva ventana
  • Las funcionalidades (básicas) pueden utilizarse sin JavaScript
