Zugänglichkeit

Inhalte und Angebote von Europeana.eu wurden so gestaltet, dass sie für alle Nutzer und Geräte zugänglich sind. Wir bemühen uns, die W3C-Leitlinien für die Zugänglichkeit von Webinhalten Version 1.0 zu erfüllen.

Unser nutzerorientiertes Designteam testet häufig die Benutzerfreundlichkeit der Navigation, der Bezeichnungen und der Konsistenz der Seite Europeana.eu. Nach dieser Prototyp-Phase werden wir außerdem eine Version für Blinde produzieren.

Einige Merkmale der guten Zugänglichkeit unserer Website umfassen:

  • Sinnvolle Seitenbezeichnungen, um die Orientierung zu erleichtern
  • Für die meisten Bilder gibt es ein sinnvolles Textäquivalent
  • Minimale Verwendung von Frames
  • Verwendung von skalierbaren Fonts
  • Verwendung von Cascading Style Sheets (CSS), um Inhalt von Präsentation zu trennen
  • der Nutzer wird informiert, wenn ein Link ein neues Fenster öffnet
  • Die (wesentlichen) Funktionalitäten sind ohne JavaScript nutzbar
