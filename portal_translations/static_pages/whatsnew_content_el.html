Τί νέο υπάρχει

Έχουμε κάνει αλλαγές, έτσι ώστε έχοντας επίσης φρέσκια όψη και αίσθημα, είναι τώρα ευκολότερο για σας να ανακαλύψετε και να εξερευνήσετε τους θησαυρούς της Europeana.

Μια από τις μεγαλύτερες αλλαγές είιναι ότι ο δικτυακός τόπος ανταποκρίνεται τώρα. Αυτό σημαίνει ότι αυτόματα υιοθετεί μια σειρά από διαφορετικά εργαλεία και μεγέθη οθονών. Έτσι είτε χρησιμοποιείτε μια οθόνη αφής έξυπνου κινητού ή μεγάλο υπολογιστή γραφείου, η δικτυακή πύλη θα προσαρμόσει την παρουσίαση έτσι ώστε η εμφάνιση να ταιριάζει πάντοτε με την οθόνη.

Η αναζήτηση καθίσταται ακόμα ευκολότερη με νέες αυτόματες προτάσεις που αναδύονται καθώς πληκτρολογείτε. Θα προβλέψουμε τί ψάχνετε και θα κάνουμε συστάσεις με βάση αυτό, πριν τελειώσετε την πληκτρολόγηση.

Η νέα ιστοσελίδα περιλαμβάνει εντονότερη εστίαση στο τεκμηριωμένο περιεχόμενο με μεγαλύτερη εμφάνιση διαφανειών εικόνων, μια χαρακτηριστική ενότητα συνεργατών και ένα μεγάλο καρουζέλ που επισημαίνει τις τελευταίες πινέζες μας στο Pinterest.

Για τους προγραμματιστές, έχουμε επίσης ανακοινώσει το Europeana API (Application Programming Interface) v2.0 με βελτιωμένη κονσόλα και νέα χαρακτηριστικά. Εγγραφείτε για τα κλειδιά API εδώ.

Ελπίζουμε να απολαύσετε τη χρήση της νέας εμφάνισης της Europeana.
