Usar a Minha Europeana

Minha Europeana é o seu espaço pessoal na Europeana.

Registe-se em Minha Europeana e pode:

  • Guardar esta pesquisa: Manter-se informado sobre um determinado assunto na Europeana guardando uma pesquisa, para que possa facilmente executá-la de novo mais tarde.
  • Adicionar uma etiqueta: Marcar os seus itens favoritos da Europeana com etiquetas com significado para si. Poderá então ver, na interface de Minha Europeana, todos os objetos que marcou, organizados pelas palavras-chave que escolheu.
  • Guardar na Minha Europeana: Guardar os seus itens favoritos para ver mais tarde.
  • Chaves API: Se é um programador registado para uma ou mais chaves API, estas serão apresentadas no tab Chaves API.

Para guardar uma pesquisa, adicionar uma etiqueta ou guardar um item, basta fazer login em Minha Europeana e clicar na acção em questão no menu.

Para ver os itens, pesquisas e etiquetas guardados, use os atalhos Minha Europeana no cabeçalho.
