Europeana: metadata schemas

There are two schemas associated with Europeana:

The Europeana Data Model (EDM) is the current reference schema for metadata in Europeana. Its namespace is http://www.europeana.eu/schemas/edm/. Documentation relating to EDM can be found at http://pro.europeana.eu/edm-documentation/.

Europeana Semantic Elements (ESE) 3.4 was Europeana's previous reference schema. Here you can find the XML schema. The associated documentation can be found at http://pro.europeana.eu/ese-documentation/.
